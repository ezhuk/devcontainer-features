# uv Dev Container Feature

Installs a uv release from the official repository.

## Example Usage

```json
"features": {
  "ghcr.io/ezhuk/devcontainer-features/uv:1": {
    "version": "0.11.7"
  }
}
```

## Options

| Option   | Type   | Default | Description           |
|----------|--------|---------|-----------------------|
| version  | string | 0.11.7  | uv version to install |
