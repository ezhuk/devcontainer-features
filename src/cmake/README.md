# CMake Dev Container Feature

Installs a CMake release from the official repository.

## Example Usage

```json
"features": {
  "ghcr.io/ezhuk/devcontainer-features/cmake:1": {
    "version": "4.3.1"
  }
}
```

## Options

| Option   | Type   | Default | Description              |
|----------|--------|---------|--------------------------|
| version  | string | 4.3.1   | CMake version to install |
